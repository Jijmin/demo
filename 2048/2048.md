# 2048游戏制作
## 游戏架构
### 制作一个简单的游戏需要什么技术？
1. 游戏逻辑
2. 技术(HTML+CSS+Javascript+jQuery)
3. 美工
4. 创意
### 游戏架构
1. MVC架构
2. M->游戏数据
3. V->HTML+CSS搭建页面得到UI
4. C->js/jQuery控制游戏主逻辑
5. 用js/jQuery实现动画效果逻辑,主逻辑控制，显示给UI界面
6. 还有一个js的支撑逻辑控制数据与主逻辑的关系
### 文件夹结构
1. index.html---->2048主页面
2. style.css---->2048样式
3. main.js---->游戏主逻辑
4. animation.js---->动画效果逻辑
5. support.js---->底层支撑逻辑
